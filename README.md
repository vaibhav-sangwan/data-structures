# Data Structures Implementation in C#

The repo demonstrates really easy to grasp implementation of Data Structures such as:
	
	- Dynamic Array
	- Linked List
	- Doubly Linked List
	- Stack
	- Queue
	- Priority Queue
	- Union Find (or Disjoint Set)
	- Segment Tree

	- Merge Sort
	- Binary Search

	All the implementations are done using C#.
